## E-Commerce Website

A full-stack E-Commerce web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).
This project provides core online shopping functionalities with secure authentication and efficient product & order management.

### Key Features:

* User authentication (JWT)
* Product listing & details
* Shopping cart functionality
* Order placement & history
* Admin dashboard for product & order management

### Tech Stack:

* **Frontend:** React.js, HTML, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **Authentication:** JWT
